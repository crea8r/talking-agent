What This Validates

This pass is about picking the right rendering contract, not finishing the whole character product.

  • The avatar is a real rigged model with a standardized humanoid skeleton.
  • Speech drives VRM mouth presets instead of hand-drawn 2D shapes.
  • Emotes and gaze remain separate from gestures so agent control stays composable.
  • Gesture choice is intentionally semantic and bounded, not arbitrary bone puppeting.
  • The browser stack stays open and local: VRM, Three.js, and local model assets.
